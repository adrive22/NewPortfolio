Relevant Projects

ShowMaps

Designed a full stack web program with React that utilizes its own created database information to implement a Google map with markers showing nearby shows in Austin for that day; significantly cutting down a users time to find a show happening near them. This program utilizes Mongo DB with MLab, Mongoose, Express, React, Node, Axios, Google Maps API with Geolocation, momentjs, and the google-maps-react npm package to help integrate the Google Maps API with React.

Album Cover Shuffle Game

Designed a simple React web game using create-react-app that uses images stored in a JSON file and shuffles them in random order everytime one of the images is clicked. If you click the same image twice, then you will be alerted and the score will be reset. This game also keeps the highest score that you have achieved overall while you are still on the same page. This  web game uses JQuery, React, and Google Fonts. It is also mobile responsive.

Eye Candy

Designed a web app to bring you an active entertaining world of giphs of your choosing using the Giphy API. You can use the permanent buttons on the site or add buttons to search the Giphy API. You can interact further with the site by starting and stopping the giphs themselves at any point. This web application utilizes Javascript, Jquery, AJAX, and the Giphy API, coupled with Semantic-UI styling and Google Fonts. This website is also mobile responsive.

Train-Scheduler

Designed a Firebase web application where users can add train information into the required fields, and math will be completed for them to tell the user when the next train will arrive and the how many minutes you have left until then. All train information input is stored and called upon from the firebase database using AJAX. This application uses Firebase, Javascript/JQuery, AJAX,  momentJS and simple CSS with Google Fonts.
